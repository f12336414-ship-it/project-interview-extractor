# Changelog

All notable changes to this project are documented here. The project follows [Semantic Versioning](https://semver.org/).

## [Unreleased]

## [1.1.0] - 2026-07-07

### Added

- Portable `help`, `chat`, `bank`, `analyze`, `resume`, `review`, and `export` skill subcommands.
- Realistic `chat` interviews with adaptive one-question turns and end-of-session feedback.
- `bank` export for a measurable 17-dimension question, answer, follow-up, and developer-learning handbook.
- Command-specific regression evaluations and validation rules.

## [1.0.0] - 2026-07-07

### Added

- Evidence-first project analysis with confirmed, inferred, and pending facts.
- Quick, deep, mock-interview, and resume-optimization modes.
- Role-aware question generation and three-level follow-up chains.
- Resume, STAR, answer-review, and scoring templates.
- Sensitive-information safeguards.
- Five evaluation scenarios and a dependency-free package validator.
- Cross-platform CI, CodeQL scanning, Dependabot, and community-health files.
