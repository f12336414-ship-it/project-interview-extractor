# Support

## Usage questions

Open a GitHub issue using the question link in the issue chooser. Include your Codex environment, invocation text, expected behavior, and a minimal anonymized example.

## Bug reports

Use the bug-report form and provide reproducible evidence. Never paste credentials, personal information, private repository content, or confidential interview material.

## Security and conduct

- Report vulnerabilities through [SECURITY.md](SECURITY.md).
- Report community-conduct concerns through [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

Support is community-based and provided on a best-effort basis; no response time or compatibility guarantee is implied.
